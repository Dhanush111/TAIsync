# First N Odds

Problem


Write a program to print the first N odd natural numbers.


Input Format

First line consists of a positive integer n


Output Format

Print the number from 1 to n space separated odd integers.


Sample Inputs & Outputs

Sample Input 1
5

Sample Output 1
1 3 5 7 9 



Sample Input 2
3

Sample Output 2
1 3 5 



Constraints
1 <= n <= 10^3
