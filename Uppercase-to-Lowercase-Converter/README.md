# Uppercase to Lowercase Converter

Problem





Converts a given string from uppercase to lowercase.





Input Format



The input consists of a single line containing a string.





Output Format



Output the converted string in lowercase.





Sample Inputs & Outputs



Sample Input 1

HELLO WORLD!



Sample Output 1

hello world!







Sample Input 2

TapACADemy



Sample Output 2

tapacademy







Constraints



1 <= String length <= 100
