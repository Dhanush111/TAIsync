# Lowercase to Uppercase Converter

Problem





Converts a given string from lowercase to uppercase.





Input Format



The input consists of a single line containing a string.





Output Format



Output the converted string in uppercase.





Sample Inputs & Outputs



Sample Input 1

hello world!



Sample Output 1

HELLO WORLD!







Sample Input 2

TapACADemy



Sample Output 2

TAPACADEMY







Constraints



1 <= String length <= 100
