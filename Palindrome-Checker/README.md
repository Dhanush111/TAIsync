# Palindrome Checker

Problem





Checks whether a given string is a palindrome or not.





Input Format



The input consists of a single line containing a string.





Output Format



Output 'Yes' if the string is a palindrome, 'No' otherwise.





Sample Inputs & Outputs



Sample Input 1

radar



Sample Output 1

Yes







Sample Input 2

TapAcademy



Sample Output 2

No







Constraints



1 <= String length <= 100
