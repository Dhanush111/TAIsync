# Multiples of 3

Problem


Write a program to print first N multiples of 3.


Input Format

First line consists of a positive integer n


Output Format

Print the required space separated number series


Sample Inputs & Outputs

Sample Input 1
7

Sample Output 1
3 6 9 12 15 18 21 



Sample Input 2
3

Sample Output 2
3 6 9 



Constraints

1 <= n <= 10^3
