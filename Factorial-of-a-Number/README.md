# Factorial of a Number

Problem





Given a non-negative integer N, find its factorial.





Input Format



An integer N (0 <= N <= 100) representing the input number.





Output Format



Print the factorial of N.





Sample Inputs & Outputs



Sample Input 1

5



Sample Output 1

120







Sample Input 2

0



Sample Output 2

1







Constraints



0 <= N <= 100
