# Multiples of 5

Problem


Write a program to print first N multiples of 5.


Input Format

First line consists of a positive integer n


Output Format

Print the required space separated number series


Sample Inputs & Outputs

Sample Input 1
5

Sample Output 1
5 10 15 20 25 



Sample Input 2
3

Sample Output 2
5 10 15 



Constraints
1 <= n <= 10^3
